  • -storeMessageForClientId:topic:data:retainFlag:qos:msgId:incomingFlag:commandType:deadline:

    Stores one new message

    • - parameter: clientId identifying the session
    • - parameter: topic the topic of the message
    • - parameter: data the message’s data
    • - parameter: retainFlag the retain flag of the message
    • - parameter: qos the quality of service of the message
    • - parameter: msgId the id of the message or zero for qos zero
    • - parameter: incomingFlag the direction of the message
    • - parameter: commandType the command of the message
    • - parameter: deadline the deadline of the message for repetitions
    • - returns: the created MQTTFlow element or nil if the maxWindowSize has been exceeded
